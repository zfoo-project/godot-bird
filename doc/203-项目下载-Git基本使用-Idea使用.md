这种方式在线比较方便，但是本地调试修改比较麻烦，不推荐

# 下载zip包本地解压使用

- 在github项目主页直接下载zip包

```
这种方式在线比较简单粗暴，但是会丢失git得提交信息，不推荐
```

# 终极解决方式，使用git下载

- 强烈推荐，会有版本记录，可以看到每次得改动记录，也方便后面切换分支
- 不需要记任何git指令，可以完全通过可视化操作工具去操作github

```
如果本地没有科学上网工具，强烈推荐FastGithub，可以看看之前得教程

后面说项目实战的时候，还会不断的去说git如何切分支，提交等等一些列使用问题
```
